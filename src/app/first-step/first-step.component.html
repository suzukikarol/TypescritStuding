Primeiros Passos

Antes de começar a codar, vamos configurar o ambiente?

Editor de texto

Existem vários tipos de editor, porém um que é gratuito e muito utilizado é o vs code da Microsoft, e mesmo que você tenho um Mac ou use Linux, pode baixar ele, que ele vai rodar e é super leve.

Node Js

O Node.js é um ambiente de tempo de execução de plataforma aberta para JavaScript do lado do servidor. O Node.js é necessário para executar o JavaScript sem um suporte ao navegador. Ele usa o mecanismo JavaScript do Google V8 para executar o código. Você pode baixar o código-fonte do Node.js. ou um instalador pré-criado para sua plataforma. O node está disponível pelo link a seguir: https://nodejs.org/en/download

Verifique no site, como proceder para instalar no sistema operacional que você utiliza!

Ao finalizar a instalação, para verificar a versão instalada, é só abrir o cmd (ou terminal de sua preferência) e digitar : node -v e teclar "Enter", em seguida aparecerá abaixo a versão que está instaldo na sua máquina.

NPM

Npm é um gerenciador de pacotes do Node, Node Package Manager,normalmente ele é instalado automaticamente junto com o node, e para verificar a sua versão, basta digitar no cmd (ou terminal de sua preferência) o seguinte comando: npm -v

Typescript

Como já baixamos o npm, podemos usar o terminal para isto, basta apenas digitar o seguinte comando: npm install -g typescript

Para verificar a versão instalada, digite : tsc -v

IDE

Aqui você pode escolher o IDE ou editor de texto que você preferir, recomendo o Vs Code, por ele ser Opensource e por ter disponibilidade para várias plataformas além da Microsoft, como a Mac ou Linux.
